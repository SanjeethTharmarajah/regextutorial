Regular Expressions Tutorial

An Introduction to Regular Expressions

Summary

In this tutorial, we will describe a regex pattern for matching email addresses. We will break down the pattern into its components and provide explanations and examples for each part. Here is the regex pattern we will be discussing:

/^([a-z0-9_\.-]+)@([\da-z\.-]+)\.([a-z\.]{2,6})$/

Anchors

Anchors in regular expressions define the position in the text where the pattern should start or end. They are represented by ^ and $. The ^ anchor specifies the start of the string, while the $ anchor indicates the end.

Example:

^startOfString
endOfString$

Quantifiers

Quantifiers determine how many times a character or group should be matched. For example, * matches zero or more, + matches one or more, and ? matches zero or one.

Example:

zeroOrMore*
oneOrMore+
zeroOrOne?

Grouping Constructs

Grouping constructs allow you to create subpatterns within your regex. They are enclosed in parentheses (). Groups can be used for capturing or non-capturing purposes.

Example:

(Cat|Dog)
(?:NonCapturing)

Bracket Expressions

Bracket expressions, also known as character classes, define a set of characters to match. For example, [aeiou] matches any vowel.

Example:

[aeiou]
[0-9]
[^a-zA-Z] (Negation)

Character Classes

Character classes are predefined sets of characters represented by escape sequences like \d (digit), \w (word character), and \s (whitespace).

Example:

\d (Digit)
\w (Word Character)
\s (Whitespace)

The OR Operator

The OR operator | allows you to specify alternatives in your regex pattern. For example, (cat|dog) matches either "cat" or "dog."

Example:

(cat|dog)
(apple|banana|cherry)

Flags

Flags in regular expressions modify how the pattern is matched. Common flags include g (global), i (case-insensitive), and m (multiline).

Example:

/abc/g (Global)
/abc/i (Case-Insensitive)
/abc/m (Multiline)

Character Escapes

Character escapes allow you to match special characters like . and * literally. To match a literal dot, use \..

Example:

\.
\*
\(
